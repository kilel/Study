Лимит времени 500/750/750/750 мс. Лимит памяти 16384/16384/16384/16384 Кб.



Дано неотрицательное целое число N. Найдите разложение N на простые множители, используя метод Шермана — Лемана.

Исходные данные
Единственная строка содержит неотрицательное целое число N, 2 ≤ N ≤ 109.

Результат
Выведите разложение N на простые множители, следуя заданному формату. Каждая строка выходного файла должна содержать пару неотрицательных целых чисел pi и qi, где pi — простой делитель N, qi — его степень в разложении N на простые множители. Простые делители N должны располагаться в выходном файле в порядке возрастания. При i ≠ j pi ≠ pj.

Input Output
12
2 2
3 1
